# Bitbucket OpenAPI Plugin

PhishLabs runs BitBucket Data-Center. Bitbucket Server plugins work with Data-Center.

Requires Java11. This is a fact.

## Testing Locally
```shell
cd openAPIViewer
atlas-run
```
### Utils
* https://<your Bitbucket Server instance>/projects?web.panels
* atlas-run
* atlas-clean
